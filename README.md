# Exercícios de Estrutura de Dados em Python

Este projeto contem as resoluções das listas de exercícios em Python.

## Arquivos

- `lista1_if_for_30.py`: lista com 30 exercícios de `if/elif/else` e `for`.
- `lista_funcoes_10.py`: lista com 10 exercícios sobre funcoes.
- `lista_vetores_30.py`: lista com 30 exercícios sobre vetores (listas).
- `bubble_sort_exercicios.py`: exercícios sobre Bubble Sort.

## Requisitos

- Python 3 instalado.

Para verificar:

```bash
python3 --version
```

## Como rodar

No terminal, entre na pasta do projeto:

```bash
cd "/Users/Seu-User/Documents/estrutura de dados"
```

Depois rode o arquivo desejado:

```bash
python3 "lista1_if_for_30.py"
python3 "lista_funcoes_10.py"
python3 "lista_vetores_30.py"
python3 "bubble_sort_exercicios.py"
```

## Como funciona cada script

### 1) `lista1_if_for_30.py`

- Abre um menu para voce escolher o exercicio de **1 a 30**.
- Digite o numero do exercício e pressione Enter.
- O programa vai pedir os dados de entrada quando necessário.

### 2) `lista_funcoes_10.py`

- Mostra menu com os 10 exercícios de funções.
- Escolha uma opcao e informe os dados solicitados.

### 3) `lista_vetores_30.py`

- Executa uma demonstracao rapida (`menu_demo`) com varios exemplos da lista.
- Se quiser, voce pode abrir o arquivo e chamar funções especificas no final do script.

### 4) `bubble_sort_exercicios.py`

- Mostra menu com os exercício de Bubble Sort (1 a 8).
- Escolha a opcao para ver o resultado de cada exercício.

## Dicas rapidas
- Para encerrar um script, use `Ctrl + C` no terminal.
